３．起動

①まずはじめにconfig.exeをダブルクリックして、いろいろと設定してください。

②設定し終わったら決定ボタンを押してconfig.exeを終了させて、

　kunio2.exeをダブルクリックしてください。それでゲームスタートです。

③一応動作速度（ＦＰＳ：ふれーむぱーせこんど）は６０で標準なので、もしそれ以下になるようでしたら、

　config.exeの「システム設定」の部分でいろいろと変更してみてください。もしかしたら速くなるかもしれません。

　(チェックを入れれば速くなると言うものではなく、パソコンによって最速の設定は異なります。)

ちなみにウインドウモードで満足した速度を出すには、多少マシンパワーが必要になるので、

フルスクリーンではＦＰＳ６０でるのに、ウインドウではいくら設定をいじっても６０出ないと言う方は

ウインドウ起動を諦めるか、僕のプログラミングの腕が上がるのを待つか、

新しいＶＡＩＯでも買ってください。
